# 아무 amoo

## 스티비 탈출을 위한 여정

- [아무 amoo](https://bit.ly/amoo_team) 뉴스레터를 stibee 밖으로 꺼내고자 하는 시도
- 발전 가능성이 보인다면 뉴스레터 SaaS가 될 지도

## 미리보기

- [Production](https://amoo.te6.in/)
- [Preview](https://preview.amoo.te6.in/)

## 개발 환경

- Node.js (20.12.0)
- pnpm (9.0.6)
- Supabase
- Vercel
